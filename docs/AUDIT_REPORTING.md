# Audit reporting

Audit events record organisation, actor/role, action, entity, before/after state, reason, reviewer comment, timestamp, correlation and result. Reports support search and printable HTML; CSV export is audited. Filters map to indexed organisation/time/action/entity fields. Normal users have no audit update/delete policy.

Approval timelines retain requested, reviewed, changes requested, resubmitted, approved and executed events. PDF export is a deployment enhancement; printable HTML supports browser Save as PDF now.
