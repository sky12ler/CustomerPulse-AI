# Devpost submission notes

**Tagline:** Turn customer evidence into explainable, approved retention and marketing action.

**Built with:** Codex, OpenAI Responses API, GPT-5.6 (environment configured), Next.js, TypeScript, Zod, Supabase, Recharts, Buffer GraphQL API, Vitest, Playwright and Vercel.

**Why it matters:** CustomerPulse connects conversation/behaviour signals to deterministic risk, evidence-linked AVO assistance and human approval. It avoids black-box churn decisions and autonomous outreach.

**Demo proof:** four synthetic scenarios, no-credential fallback, permanent imports/documents, AVO evidence validation, approval-gated outreach/publishing, privacy controls and correlated audit history. Add the Codex `/feedback` Session ID, GitHub URL, Vercel URL and video URL before submission.

**Human decisions:** staff verify evidence and edit drafts; managers decide actions/campaigns/overrides/exports; employees explicitly execute private outreach. AVO cannot approve, send, publish or override.
