## Final acceptance caveats

The no-credential demo intentionally keeps workflow changes in browser memory; reloading restores seeded demo state. The Supabase schema and reset loader are ready, but the current UI does not persist interactive demo mutations to a deployed Supabase project. A real Supabase project, live Buffer publishing, and a real GPT-5.6 network response could not be tested without credentials. Those limitations are reported rather than represented as passing external integrations.
