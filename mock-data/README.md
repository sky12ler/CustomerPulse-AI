# CustomerPulse AI mock data

All records are fictional and labelled **Synthetic Demo Data**. Domains use `.example` or `.invalid`; phone numbers are non-production demo values.

The files exercise four scenarios: `CUS-1001` strategic risk, `CUS-1002` growth opportunity, the North food-and-beverage decline, and `CUS-1004` successful recovery. CSV/JSON files can be imported from **Data Imports**. PDFs are approved grounding sources; `existing-campaign.png` is a synthetic campaign asset.

Keep these files permanently in `/mock-data`; the reset script reloads them and never removes them.

For an isolated Imported Workspace regression with multiple risk, consent and evidence cases, use the ordered pack in [`scenarios/UPLOAD_MANIFEST.md`](scenarios/UPLOAD_MANIFEST.md).
